# Merge_SEU
合成东南大学(合成大西瓜魔改版)

这是一个合成大西瓜的魔改项目，贴图设置为东南大学等高校校徽，最大与最小水果均为东南大学。

点击屏幕释放水果，点击右上东南大学图标切换水果。
